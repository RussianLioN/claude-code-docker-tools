# Внедрение Универсальной Синхронизации (Trae + All AI IDEs)

Мы адаптируем скрипт `sync-ai-ide-files.sh` для поддержки Trae IDE, превращая его в единый механизм управления контекстом для всех AI-ассистентов.

## 1. Модификация скрипта `scripts/sync-ai-ide-files.sh`
Внести следующие изменения:
1.  Добавить `.trae/rules/00-system-manifest.md` в список отслеживаемых файлов.
2.  Добавить переменную `TRAE_YAML_HEADER` с конфигурацией (alwaysApply: true).
3.  Добавить логику ветвления (if/else) для обработки Trae-файлов:
    *   **Write:** Добавлять заголовок при записи в `.trae/...`.
    *   **Read:** Удалять заголовок при чтении из `.trae/...` (чтобы не дублировать его в другие IDE).

## 2. Исполнение (Deployment)
1.  Запустить `./scripts/sync-ai-ide-files.sh`.
    *   *Результат:* Скрипт возьмет самый свежий существующий файл (например, `CLAUDE.md`) и создаст на его основе правило для Trae.

## 3. Проверка и Фиксация
1.  Проверить содержимое созданного `.trae/rules/00-system-manifest.md`.
2.  Закоммитить изменения скрипта и нового правила в Git: `feat(devops): add Trae support to AI sync system`.
